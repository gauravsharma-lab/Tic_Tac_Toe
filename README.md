Tic-Tac-Toe Game
Overview:
A simple and interactive Tic-Tac-Toe game built using HTML, CSS, and JavaScript. Players take turns marking X and O on a 3Ã—3 grid, with logic to detect wins, draws, and reset the game.
Features:
-  Two-player turn-based gameplay
-  Win and draw detection
-  Game reset functionality
-  Clean UI with responsive design
Tech Stack:
- HTML5
- CSS3
- JavaScript (DOM manipulation & game logic)
How to Play: Click on any empty cell to place your mark. The game announces the winner or a draw when the board is full.
